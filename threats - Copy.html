1. Malware

Malware attacks are the most frequent risks to cyber security. Malware is referred to as harmful software that infiltrates a computer after a user clicks on a malicious link or email, such as viruses,worms and so on. Malware will destroy the system, collect private data, and more once it has gained access to it.

2. SQL Injections

A Structured Query Language (SQL) injection attack occurs when hackers upload malicious SQL scripts in an attempt to gain access to a database. After being successful, the malicious actor can read, modify, or delete data stored in the SQL database.

3.Ransomware

Ransomware is a type of malware attack in which the attacker locks or encrypts the victim's data and demands a ransom in exchange for not disclosing or blocking access to the data.

4.Phishing

Phishing is Malicious emails sent by hackers pretend to be from reliable sources. The malicious link in the email is then deceived into being clicked by the user, which leads to the installation of malware or the exposing of private data like login credentials and credit card numbers.

5. Spyware

Spyware is a type of malware that allows hackers to gain unauthorized access to data, including sensitive information such as payment details and credentials. Spyware can infiltrate mobile phones, desktop applications, and web browsers.

6. Worms

Worms is a kind of malware targets backdoors and vulnerabilities to obtain unauthorized access to operating systems. The worm can launch a number of attacks after installation, such as Distributed Denial of Service (DDoS).

7. Pharming

Pharming is a type of online fraud in which a cybercriminal installs malicious software on a server or computer. Users are tricked into providing personal information on a bogus website by the code that automatically redirects them there.
